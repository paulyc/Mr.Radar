RSL_get_gr_slantr_h

Synopsis

#include "rsl.h"
void RSL_get_gr_slantr_h(Ray *ray, int i, float *gr, float *slantr, float *h)

Description

Given a Ray ray and the range index i, calculate the ground range, slant range, and height; gr, slantr and h respectively.

Return value

None.
