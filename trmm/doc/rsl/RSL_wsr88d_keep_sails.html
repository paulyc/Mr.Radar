RSL_wsr88d_keep_sails

Synopsis

#include "rsl.h"
void RSL_wsr88d_keep_sails(void);

Description

Call this function to keep the low-level split cut found in the middle of the volume scan when SAILS mode is in effect. By default this split cut is removed.
Call RSL_wsr88d_keep_sails before calling RSL_anyformat_to_radar.

Return value

None.
