RSL_wsr88d_asis

Synopsis

#include "rsl.h"
void RSL_wsr88d_asis(void);

Description

Call this function to specify that all sweeps are to be stored as read. That is, don't combine split-cuts by elevation or remove short-range reflectivity in Doppler cuts. Also keep the SAILS split cut if present. Call RSL_wsr88d_asis before calling RSL_anyformat_to_radar.

Return value

None.
